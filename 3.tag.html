태그

태그는 html을 만들기 위한 도구로 < >,< / >이 하나로 구성되어있어서 괄호안에 사용할 도구를 집어넣고 그 사이에 표현할 것을 적어 웹에 표시한닷. 하지만 < > 만 존재하는 태그도 있다는 것!
이 밑에는 주로 사용되는 인기가 많은 태그들을 보여준다

  • head 는 한마디로 웹을 설명하는 부분을 나타내는 태그
  • body 는 웹의 본문을 담당하는 태그
  • html 은 head와body를 덮고있는 태그.시작과 끝을 담당
  • title 은 웹의 제목으로 브라우저 탭에 글씨가 보여짐
  • meta 는 웹 페이지의 보이지 않는 정보를 제공하는데 쓰이는 태그
  • div 는 레이아웃을 나눌 때 사용 주로 css와 함께 쓰임
  • a 는 하이퍼링크로 말그대로 링크태그
  • script 는 자바스크립트와 같은 클라이언트 사이드 스크립트(client-side scripts)를 정의할 때 사용
  • link 는 모루갰당
  • p는 paragraph로 한 단락을 의미 한다. 단락과 단락사이를 구분할수 있다.
  • li 는 list로 목차를 나타낼때 쓴다 li는 ul,ol과 같이 쓰인다.
  • ul 은 unordered로 목차를 구분해줄때 쓰인다. li와 같이 쓰임. *ol은ordered list 숫자로 정렬

나머지는 추가로 알게된 것들을 적어보고자 한다.

  • !doctype html 은 이 웹이 html로 만들어 졌다는 것을 표시하기위해 html태그 위에 !doctype html 태그를 넣는다
  • src 는source 로 이미지를 불러올때 사용하는듯 사이트주소나 폴더에있는 이미지를 불러
  • width 는 이미지의 폭을 키울때 숫자와 퍼센트를 이용해 사용한다. ex)img src="~~"widht="00%"
  • input 은 뭔갈 추가할 때 쓰는거같은데 이런식??
